## Installation

Requires [Node.js](https://nodejs.org/) 
Requires[Express](https://www.npmjs.com/package/express)


## Add project 

```
git clone https://github.com/kaanavcix/PsikozBackendTs
```

then

```sh
npm install 
```
run with ts-node

```
npx ts-node ./src/index.ts
```
